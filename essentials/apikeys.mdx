<Info>If you don't have an Ultravox account, you can sign up at https://app.ultravox.ai. All new accounts get 30 free minutes of call time.</Info>

Your first key must be created using the Ultravox console. Once you have created and API key, you can use the REST API to [create](/api-reference/accounts-apikeys/apikeys-post), [update](/api-reference/accounts-apikeys/apikeys-patch), and [delete](/api-reference/accounts-apikeys/apikeys-delete) API keys.
<Steps>
  <Step title="Sign-In">
    Make sure you are signed into you account at https://app.ultravox.ai
  </Step>
  <Step title="Go to Settings">
    In the left nav, click on Settings. Or navigate to https://app.ultravox.ai/settings/.
    <img className="block h-64 dark:hidden" src="/images/screenshots/settings-light.png" />
    <img className="hidden h-64 dark:block" src="/images/screenshots/settings-dark.png" />
  </Step>
  <Step title="Click 'Generate New Key'">
    In the API Keys sections, click on "Generate New Key".
    <img className="block h-64 dark:hidden" src="/images/screenshots/generatenewapikey-light.png" />
    <img className="hidden h-64 dark:block" src="/images/screenshots/generatenewapikey-dark.png" />
  </Step>
  <Step title="Name the Key and Save">
    Create a name for your new API key and create it. Make sure you save the key in a secure secrets manager.
    <img className="block h-64 dark:hidden" src="/images/screenshots/newapikey-light.png" />
    <img className="hidden h-64 dark:block" src="/images/screenshots/newapikey-dark.png" />
  </Step>
</Steps>
