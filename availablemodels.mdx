The Ultravox API currently provides the following models.

| Model | Description |
|---|---|
|fixie-ai/ultravox| This is an alias that points to `ultravox-70B`. The default if no model is specified when creating a call with the API. |
|fixie-ai/ultravox-70B  | Based on Llama 3.3 70B. Supports tools. |
|fixie-ai/ultravox-8B| Based on Llama 3.3 8B. Not recommended for most use cases. Tools are unlikely to work. |

## Using Models
Each model can be used when [creating a call](./api/calls/#create-call) via the API. For example:

```javascript
// Example request body to create an Ultravox call
{
  "systemPrompt": "You are a helpful assistant...",
  "model": "fixie-ai/ultravox-70B"
}
```
